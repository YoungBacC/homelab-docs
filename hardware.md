# Hardware
The main box is a Dell Optiplex 3046 Small Form Factor (SFF). I have made a few upgrades in terms of the base hardware.

- The CPU is a intel i5-6500 3.20 GHz.
- The RAM installed is one stick of GSkill 16GB 3200 MHz (this is the limit on the optiplex)
- There are 3 drives: The OS drive is a 128 GB Sata SSD, 1TB Sata HDD, 500 GB Sata HDD.

As a note, I have 3 drives installed and this is the max. So if I want more space, I would need to upgrade my MOBO for more sata ports and case for more drive bays. OR I can upgrade the drives themselves. I would probably go with the former since this would leave room for expansion and redundancy drives.
