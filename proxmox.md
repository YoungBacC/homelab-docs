# Proxmox
Proxmox is installed on the 128 GB sata SSD. It is running on IP address 192.168.1.241 on port 8006. 

Once Proxmox was running, I set up ZFS on both the free drives (not the SSD) for bulk storage. They are named bulk-1tb and backup-500gb. They are named this way because I want to use the 500gb as a back up drive.
